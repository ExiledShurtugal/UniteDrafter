# Decrypter tests

This folder contains automated tests for the decrypter.

## Run from repo root

```
dotnet clean
dotnet test tests/Decrypter/DecrypterTests.csproj --logger "console;verbosity=detailed"
```

Expected outcomes:

1. xUnit test results in the console (`Passed` / `Failed`).
2. Decrypted fixture artifacts written to:
   - `tests/Decrypter/TestResults/decrypted_blastoise.json`

That file is generated by a fixture-based test and lets you manually inspect decrypted output.
